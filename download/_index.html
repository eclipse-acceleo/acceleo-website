Install with the update site

The easiest (and quickest) way of installing Acceleo is to use the Helios update-site.

  1. Install your preferred Eclipse distribution from the Eclipse download page.
  2. Go to Help => Install New Software...
  3. In the upper part, select the Helios Update site, then tick "Acceleo SDK" in the lower panel, such as shown below.
  4. Click Finish, read and accept the Eclipse Public License, and watch Acceleo install itself along with its dependencies.

Download the latest Acceleo

The latest versions of Acceleo are the 3.x stream builds. Check them out on the project's download page.

Download maintenance releases of Acceleo

Maintenance builds of Acceleo 2.x are still available on the maintenance versions download page.

Install with the Amalgamation project

  1. Download the latest EPP modeling bundle
  2. Use the modeling discovery button to install Acceleo

Here you go, it doesn't get any easier!
Note that whatever your installation procedure is, examples are available from the menu right-click => New => Examples => Acceleo Plug-ins.
